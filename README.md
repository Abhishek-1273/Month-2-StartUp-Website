# 🚀 StartUp - Multi-Page Business Website

This is a fully responsive and visually appealing multi-page website designed for startups, tech teams, or digital agencies. Built using HTML, CSS, and little bit JavaScript, it includes multiple pages and features like:

## 🧩 Pages Included
- *Home* – Introduction with hero section and company overview.
- *About* – Team profiles with roles and descriptions.
- *Blog* – Placeholder for blog content and updates.
- *Contact* – Contact form or details for reaching out.
- *Services* – Highlights what your startup offers.
- *Pricing* – Clear breakdown of service plans.
- *Strategy* – Explains your approach and working method.

## 📌 Key Features
- Clean and modern layout
- Mobile responsive design
- Scroll animations and section reveals
- Organized folder structure with reusable components
- Font from Google Fonts (Poppins)
- Uses icons/images for branding

## 📁 Tech Stack
- HTML5
- CSS3 (Flexbox & custom styling)
- JavaScript (little bit)
